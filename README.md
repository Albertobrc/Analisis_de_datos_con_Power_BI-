# Analisis_de_datos_con_Power_BI-

## 📌 Descripción

Este repositorio contiene una colección de dashboards desarrollados en Power BI como parte de mi proceso de aprendizaje y aplicación práctica de los conocimientos adquiridos durante una especialización en Microsoft Power BI Data Analyst realizada a través de Coursera.

El objetivo del proyecto es demostrar competencias en análisis de datos, modelado de datos, creación de métricas con DAX y diseño de reportes interactivos orientados a la toma de decisiones.

## 🎯 Objetivos

* Transformar datos en información útil para el negocio.
* Diseñar dashboards interactivos y visualmente atractivos.
* Aplicar buenas prácticas de modelado de datos.
* Crear indicadores clave de desempeño (KPIs).
* Facilitar el análisis mediante visualizaciones dinámicas.

## 🛠️ Herramientas y Tecnologías

* Power BI Desktop
* DAX (Data Analysis Expressions)
* Power Query
* Microsoft Excel
* Modelado de Datos

## 📊 Contenido del Proyecto

Los dashboards incluidos en este repositorio abarcan diferentes escenarios de análisis, aplicando conceptos como:

* Limpieza y transformación de datos.
* Creación de relaciones entre tablas.
* Diseño de modelos semánticos.
* Desarrollo de medidas y columnas calculadas con DAX.
* Uso de segmentadores, filtros y navegación interactiva.
* Creación de KPIs y métricas de negocio.
* Visualización efectiva de datos.
